# Hairdesign Barbara Leitner

Statische, responsive Website für Hairdesign Barbara Leitner in Krieglach. Die Seite besteht aus reinem HTML, CSS und JavaScript und benötigt keinen Build-Schritt.

## Projektstruktur

```
├── index.html
├── unser-salon.html
├── leistungen.html
├── team.html
├── kontakt.html
├── impressum.html
├── datenschutz.html
├── css/style.css
├── js/script.js
└── images/
```

## Lokal öffnen

Das Projekt im VS Code öffnen und mit der Erweiterung **Live Server** starten. Die Startseite ist `index.html`.

## Veröffentlichung mit GitHub und Vercel

1. Änderungen in Git committen und nach `main` pushen.
2. Das GitHub-Repository einmalig in Vercel importieren.
3. Vercel erkennt die statische Website automatisch und veröffentlicht jeden Push auf `main`.
4. Die eigene Domain in Vercel unter **Settings → Domains** verbinden.

## Vor dem Livegang prüfen

- In `kontakt.html` die echte Formspree-ID in der Formular-URL einsetzen.
- Preisangaben in `leistungen.html` mit dem Salon abstimmen.
- Öffnungszeiten final bestätigen.
- Im Impressum die zuständige Gewerbebehörde und Firmenbuchnummer ergänzen.
- Datenschutzerklärung nach Aktivierung externer Dienste rechtlich prüfen lassen.
